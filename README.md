##Project Overview

This project focuses on analyzing retail sales data using SQL to extract meaningful insights related to sales performance, customer behavior, and product trends.

* GitHub

##Dataset Description

The dataset contains information about:

* Orders
* Customers
* Products
* Sales and revenue

##Key SQL Concepts Used

* SELECT, WHERE, ORDER BY
* GROUP BY, HAVING
* JOINS (INNER, LEFT)
* Subqueries
* Aggregate Functions (SUM, AVG, COUNT)
* Date Functions

##Analysis Performed

* Total and monthly sales analysis
* Top-selling products
* Customer-wise revenue contribution
* Profit and quantity analysis

##Sample Query

```sql
SELECT product_name, SUM(sales) AS total_sales
FROM sales
GROUP BY product_name
ORDER BY total_sales DESC;
```
